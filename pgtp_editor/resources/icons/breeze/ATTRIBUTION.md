# Breeze icon attribution

The SVG icons in this folder are from the KDE **breeze-icons** project:

https://github.com/KDE/breeze-icons

© KDE contributors. Licensed under **LGPL-3.0-only** (see
`LICENSE-LGPL-3.0.txt` in this folder).

The files are unmodified upstream Breeze SVGs. PGTP Editor recolors them only
at runtime (substituting a concrete fill for the `currentColor` /
`.ColorScheme-Text` mechanism) so the toolbar icons stay legible against both
the light and dark themes; the vendored files on disk are byte-for-byte
upstream.

Icons used:

- `document-open.svg`
- `document-save.svg`
- `edit-undo.svg`
- `edit-redo.svg`
- `edit-find.svg`
- `dialog-ok-apply.svg`
- `run-build.svg`
